Файл "gitignore" визначає навмисно невідстежувані файли, які Git має ігнорувати. Файли, які вже відстежуються Git, не впливають. Кожен рядок у gitignoreфайлі визначає шаблон. Вирішуючи, чи ігнорувати шлях, Git зазвичай перевіряє gitignoreшаблони з кількох джерел із таким порядком пріоритету, від найвищого до найнижчого
